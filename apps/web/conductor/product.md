# Initial Concept

**Project Name:** Treno Chat Local
**Description:** A chat application that allows passengers to chat locally.
**Key Components:**
*   **Frontend:** Next.js 16 (React 19), TypeScript, Tailwind CSS, Radix UI.
*   **Backend:** Node.js with Express.js, Socket.IO for real-time communication.
*   **Database:** PostgreSQL with Prisma ORM.
*   **Mobile:** Hybrid mobile app using Capacitor (Android and iOS).
